# Fintech

## Overview
This project was developed as part of the **PIDEV – 3rd Year Engineering Program** at **Esprit School of Engineering** (Academic Year 2025–2026).

It is a full-stack desktop application that enables users to manage financial data and perform various fintech-related operations through an intuitive interface.

## Tech Stack
- **Java**
- **JavaFX**
- **MySQL**

Developed at **Esprit School of Engineering – Tunisia**
